Aplikasi Pencatatan Keuangan (Tema Gelap/Terang)
===============================================

Isi paket:
- index.html
- styles.css (mendukung dark & light theme)
- script.js (menyimpan preferensi tema di localStorage)
- manifest.json
- sw.js (service worker)
- icons/

Fitur baru:
- Toggle tema gelap/terang (disimpan di localStorage)
- Tombol Pasang (Install) tetap ada
- Semua fitur PWA sebelumnya tetap ada

Catatan:
- Untuk uji service worker, jalankan di server lokal atau hosting (beberapa browser tidak menjalankan SW via file://).
- Ganti ikon di folder icons/ jika ingin tampilan lebih profesional.
